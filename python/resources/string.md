### 字符串前面的字符含义

| 符号  | 含义             | 示例         |
|-----|----------------|------------|
| r   | 不转移字符串，输出原本的内容 | r'c:\temp' |
| b   | 二进制字符串         | b'Abc'     |



#### r'string'

想要了解下面的两段内容为什么不一样之前必须知道什么叫转义符。
```python
print('c:\test\a.txt')
print(r'c:\test\a.txt')
```
结果
```
c:      est.txt
c:\test\a.txt
```
